Class SortedMap<KeyType, ValueType>

The SortedMap object holds key-value pairs and automatically reorders them by values in descending order.

Type parameters

  • KeyType

    The type of elements to be stored as keys.

  • ValueType: number

    The type of elements to be stored as values. Should extend number type.
